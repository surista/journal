## üéµ Audio Tools

### Uploading Audio Files
1. **File Requirements**
   - Format: MP3 only
   - Size: Maximum 20MB
   - Quality: Any MP3 bitrate supported

2. **Upload Process**
   - Navigate to Audio Tools tab
   - Click "üéµ Upload Audio File"
   - Select your MP3 file
   - Wait for processing (shows waveform when ready)

### Audio Playback Controls
- **‚ñ∂Ô∏è Play**: Start playback from current position
- **‚è∏Ô∏è Pause**: Pause at current position
- **‚èπÔ∏è Stop**: Stop and return to beginning
- **Timer Sync**: Automatically sync with practice timer

### Speed Control
1. **Adjust Tempo**
   - Use slider or +/- buttons
   - Range: 50% to 150% of original speed
   - Fine control: ¬±1% increments

2. **Pitch Preservation**
   - ‚úÖ **Enabled**: Maintains original pitch (recommended)
   - ‚ùå **Disabled**: Pitch changes with speed (chipmunk effect)

### Pitch Shifting
1. **Semitone Adjustment**
   - Range: ¬±12 semitones (full octave)
   - Half-step precision for fine tuning
   - Perfect for playing along in different keys

2. **Quick Presets**
   - Common transpositions (P4‚Üì, m3‚Üì, M2‚Üì, etc.)
   - One-click setup for standard tuning differences

### A-B Loop Practice
1. **Set Loop Points**
   - **üìç Set Loop Start (A)**: Mark beginning of section
   - **üìç Set Loop End (B)**: Mark end of section
   - Visual markers appear on waveform

2. **Loop Options**
   - **Infinite Loops**: Continuous repetition
   - **Counted Loops**: Set specific number (1-20 times)
   - **Reset Counter**: Start count over

3. **Loop Best Practices**
   - Start with short sections (4-8 bars)
   - Practice slowly, then gradually increase tempo
   - Focus on trouble spots with tight loops

### Audio Quality Settings
1. **Quality Modes**
   - **Low**: Fast processing, basic quality
   - **Medium**: Balanced performance and quality
   - **High**: Best quality, more CPU intensive

2. **Adaptive Quality**
   - Automatically adjusts based on device performance
   - Prevents audio glitches on slower devices

### Saving Audio Sessions
1. **Save Configuration**
   - Click "üíæ Save This Session"
   - Enter descriptive name
   - Add notes about what you're practicing

2. **Saved Session Features**
   - Stores loop points, speed, pitch settings
   - Quick recall for future practice
   - Organized by audio file

## üìù Practice Logging

### Accessing Practice Forms
Practice logging forms are available on multiple tabs for convenience:
- **Practice Tab**: Primary location with expanded form
- **Audio Tools Tab**: For logging audio-based practice
- **Metronome Tab**: For rhythm-focused practice sessions
- **Repertoire Tab**: Quick logging from song cards

### Logging a Practice Session
1. **Required Information**
   - **Practice Area**: Select from dropdown (Scales, Chords, Songs, etc.)
   - **Duration**: Automatically filled from timer

2. **Optional Details**
   - **BPM/Tempo**: Enter beats per minute or percentage
   - **Musical Key**: Select key if relevant
   - **Notes**: Describe what you worked on

### BPM vs Percentage Mode
- **BPM Mode**: Enter actual beats per minute (40-300)
- **Percentage Mode**: Enter tempo as percentage (e.g., 75% = slower)
- Toggle between modes with üîÑ button

### Practice Areas
Choose from predefined areas:
- **Scales**: Major, minor, pentatonic, modes
- **Chords**: Open chords, barre chords, progressions
- **Arpeggios**: Broken chord patterns
- **Songs**: Complete song practice
- **Technique**: Picking, fingering, advanced techniques
- **Theory**: Music theory study
- **Improvisation**: Creative playing and soloing
- **Sight Reading**: Reading sheet music
- **Ear Training**: Interval and chord recognition
- **Rhythm**: Timing and groove practice

### Session Notes Tips
- Be specific: "Worked on Am pentatonic scale, 80 BPM"
- Track progress: "Increased tempo from 100 to 110 BPM"
- Note challenges: "Struggled with string bending accuracy"
- Record breakthroughs: "Finally nailed that difficult passage!"

### Form Behavior
- **Auto-save**: Form state automatically saved as you type
- **Collapsible**: Click header to collapse/expand form
- **Persistent**: Form state persists across tab switches
- **Synchronized**: All forms share the same data and timer

## üîß Troubleshooting

### Navigation Issues

#### Calendar Won't Load
**Symptoms**: Calendar button doesn't work or page shows error
**Solutions**:
- Refresh the page and try again
- Clear browser cache and cookies
- Check browser console for error messages
- The app has multiple fallback strategies for navigation
- Try using a different browser

#### Dashboard Navigation Problems
**Symptoms**: Can't return to Dashboard from Calendar
**Solutions**:
- Use "‚Üê Back to Dashboard" button in Calendar
- Refresh page if navigation seems stuck
- Check if timer is still running (it should persist)
- Clear browser cache if problems persist

### Repertoire Issues

#### Add Song Button Not Working
**Symptoms**: Clicking "Add Song" doesn't open modal
**Solutions**:
- Refresh the page and try again
- Check browser console for JavaScript errors
- Clear browser cache and reload
- Try clicking directly on the button text

#### Modal Not Appearing
**Symptoms**: Button responds but modal doesn't show
**Solutions**:
- Check if modal is behind other content (scroll or zoom out)
- Disable browser extensions that might interfere
- Try different browser
- Clear cache and hard refresh (Ctrl+F5)

### Common Issues

#### Audio Won't Load
**Symptoms**: File upload fails or no waveform appears
**Solutions**:
- Ensure file is MP3 format
- Check file size (must be under 20MB)
- Try different MP3 file
- Refresh browser and try again

#### Timer Not Syncing
**Symptoms**: Timer doesn't start with audio playback
**Solutions**:
- Enable sync checkbox: "üîó Sync" on timer
- Refresh page and try again
- Check that timer is visible and functional
- Verify audio is actually playing

#### Data Not Saving
**Symptoms**: Practice sessions disappear after closing browser
**Solutions**:
- Check browser storage permissions
- Clear browser cache and cookies
- Export data as backup before troubleshooting
- Try different browser

#### Performance Issues
**Symptoms**: Slow response, audio glitches, or freezing
**Solutions**:
- Close other browser tabs
- Lower audio quality setting in Audio Tools
- Clear old practice data via export/import
- Restart browser
- Check available device memory

#### PWA Installation Problems
**Symptoms**: Can't install as app
**Solutions**:
- Ensure using HTTPS or localhost
- Try different browser (Chrome/Edge work best)
- Clear browser cache
- Check if already installed

### Browser-Specific Issues

#### Chrome/Edge
- **Audio Context**: May require user interaction to start
- **Storage**: Generally excellent compatibility
- **Performance**: Best performance for audio processing
- **Navigation**: Most reliable for all features

#### Firefox
- **Audio Processing**: May have slight quality differences
- **PWA**: Limited installation options
- **Storage**: Good IndexedDB support
- **Navigation**: Generally reliable

#### Safari
- **Audio Limitations**: Some Web Audio API restrictions
- **Mobile Safari**: Background limitations may affect timer
- **Storage**: More restrictive storage policies
- **Navigation**: May require page refresh for some features

### Getting Help
1. **Check Browser Console** (F12) for error messages
2. **Try Demo Account** to isolate data-related issues
3. **Export Data** before troubleshooting to prevent loss
4. **Test Different Browser** to isolate browser-specific issues
5. **Create GitHub Issue** with detailed problem description

### Performance Optimization
- **Regular Cleanup**: Export and clear old data monthly
- **Quality Settings**: Use "Low" audio quality on older devices
- **Browser Updates**: Keep browser updated for best performance
- **Hardware**: More RAM helps with large audio files and navigation

### Recovery Procedures
If the app becomes unresponsive:
1. **Refresh Page**: Often resolves temporary issues
2. **Clear Cache**: Use browser settings to clear cache
3. **Export Data**: Save your data before major troubleshooting
4. **Reinstall PWA**: Uninstall and reinstall PWA if installed
5. **Different Browser**: Try in different browser as fallback
